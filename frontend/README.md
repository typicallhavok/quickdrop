# Quickdrop Frontend

The frontend graphical user interface for Quickdrop, built with [Tauri](https://tauri.app/), [SvelteKit](https://kit.svelte.dev/), and TypeScript.

## Development

To get started with development:

1. Install Node.js dependencies:
   ```bash
   npm install
   ```
2. Start the Tauri development window:
   ```bash
   npm run tauri dev
   ```
